Currently Building

Tally

Tally is a constraint-based word puzzle game where players must uncover a hidden word by reasoning about cost.

The game presents a grid of letters, where each row has an associated value. Players can form any valid word using the grid, but instead of being told the target cost, they must infer it through feedback. After each guess, the game indicates whether the word is over, under, or exactly at the hidden budget—forcing players to iterate using both vocabulary and deduction.

This creates a unique gameplay loop that blends word formation with logical inference, rewarding players for identifying patterns in both language and cost structure.
